# Разбиения на различные нечетные части

В работе исследуется количество разбиений $r(n,k)$ натурального числа $n$ ровно на $k$ различных нечетных частей. Реализованы три алгоритма вычисления $r(n,k)$, два из которых основаны на методе динамического программирования.

## Алгоритмы

| Алгоритм | Описание | Временная сложность | Пространственная сложность |
| :--- | :--- | :--- | :--- |
| `distinct_odd_parts_naive` | Прямая рекурсивная реализация соотношения. Экспоненциальное время, пригоден только для очень малых $n$. | - | - |
| `distinct_odd_parts` | Динамическое программирование на основе рекуррентного соотношения для разбиений на различные нечетные части. | $O(n \cdot k)$ | $O(n \cdot k)$ |
| `restrict_parts` | Использует биекцию с функцией $P(m,k)$ (разбиения с частями $\le k$). **Наиболее производительный на практике**. | $O(n \cdot k)$ | $O(n)$ |

Первый алгоритм (`distinct_odd_parts_naive`) оставлен для демонстрации экспоненциального роста рекурсивных вызовов и в практических вычислениях не используется. Третий алгоритм (`restrict_parts`) работает с одномерным массивом и выполняет меньше операций на итерацию, поэтому на практике он оказывается самым быстрым при больших $n$

## Применение

Разработанные алгоритмы позволили провести численную проверку гипотезы Каргаполова о рангах группы центральных единиц целочисленного группового кольца знакопеременной группы $A_n$, связанных с разбиениями на различные нечетные части при дополнительных ограничениях. Вычисления до $n = 100 000$ подтверждают гипотезу с относительной погрешностью менее $0.1\%$.
